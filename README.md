# Suzy Travel Blog

A static travel blog featuring destination stories, guides, and photo-rich article pages.

## Project Structure

- `index.html` - Homepage
- `blog.html` - Blog listing page
- `about.html` - About page
- `articles/` - Individual destination articles

## Local Preview

Open `index.html` in a browser, or run a simple local server:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000.

## Deployment

This project is published with GitHub Pages from the `main` branch (root).
